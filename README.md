# Student Collection Manager

A simple Python program to manage student information (easy enough for
non-coders to understand).

##  What This Program Does

This program works like a mini student database. It lets you: - Add new
students\
- Display all student records\
- Update student information\
- Delete a student\
- Show all subjects offered\
- Exit the system

##  How to Run the Program

1.  Install Python.

2.  Save the program file as:\
    `collection manipulator.py`

3.  Run using:

        python "collection manipulator.py"

##  Features

### 1️⃣ Add Student

Enter: - Student ID\
- Name\
- Age\
- Date of Birth\
- Grade\
- Subjects (comma-separated)

### 2️⃣ Display All Students

Shows all stored students.

### 3️⃣ Update Student

Update information using student ID.

### 4️⃣ Delete Student

Remove a student using student ID.

### 5️⃣ Display Subjects Offered

Shows all unique subjects from all students.

### 6️⃣ Exit

Closes the program.
